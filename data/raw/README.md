## Required Data Files

Place both files in this directory before running main.py.

1. **ratings_Electronics.csv**
   Download from: https://www.kaggle.com/datasets/saurav9786/amazon-product-reviews

2. **meta_Electronics.json.gz**
   Download from: https://cseweb.ucsd.edu/~jmcauley/datasets/amazon_v2/ (2018 electronics category)
